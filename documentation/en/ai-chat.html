AI Chat

Overview

Moe 2.0 includes an AI chat assistant powered by Claude. Double-click the mascot to open a speech bubble where you can chat with Moe. Moe can answer questions, help with tasks, and control your Haiku system using available tools.

Getting Started

  1. Get a Claude API key from console.anthropic.com
  2. Right-click the mascot → Settings → AI tab
  3. Paste your API key and click Save
  4. Double-click the mascot to start chatting

Speech Bubble

The chat bubble appears near the mascot and positions itself automatically based on available screen space. It shows the conversation history with colored text:

  • Blue - your messages
  • Green - Moe's responses
  • Red - errors
  • Brown - tool usage

The bubble hides automatically after 30 seconds of inactivity, on workspace change, or when you press Escape.

Voice (Text-to-Speech)

Moe can read responses aloud. Go to Settings → Voice tab:

  • Read responses aloud - enable/disable TTS
  • Voice - choose between espeak (fast, robotic) or Piper neural voices (natural, slower)
  • Speed - adjust speech rate

Piper voices require the pipertts package and voice models installed in ~/config/non-packaged/data/pipertts/models/. Language is detected automatically from the response.

Tool Integration

When a Pippo MCP server is running at the configured URL (default: http://127.0.0.1:2607/mcp), Moe can use system tools to control Haiku on your behalf. For example:

  • List and launch applications
  • Read and write files
  • Control windows and UI elements
  • Query system information

Settings

Right-click the mascot → Settings to configure:

AI Tab

  • API Key - your Claude API key (stored locally)
  • Model - Claude Sonnet 4, Opus 4, or Haiku 3.5
  • MCP URL - Pippo MCP server address
  • System prompt - customize Moe's personality

Mascot Tab

  • Change mascot - select a different mascot image
  • Wink/Polling/Redraw - animation timing settings
  • Debug frame - show mascot window boundaries

Voice Tab

  • Read responses aloud - enable TTS
  • Voice - espeak or Piper neural voices
  • Speed - speech rate

Conversation Memory

Moe remembers your conversation history across sessions. The history is saved in ~/config/settings/Kamnagi/Moe/chat_history and loaded automatically on startup (up to 20 message pairs).

Moe is also aware of which application and window you are currently using, allowing for contextual responses.
